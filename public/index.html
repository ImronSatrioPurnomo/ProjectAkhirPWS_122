  • Dokumentasi pakai standar OpenAPI 3 (Swagger)

Contoh request (curl)

# 1) bikin API key dulu lewat portal (UI) atau:
curl -X POST http://localhost:3000/portal/keys \
  -H "Content-Type: application/json" \
  -d '{"name":"Bro","email":"bro@example.com"}'

# 2) pakai key untuk create movie (protected)
curl -X POST http://localhost:3000/v1/movies \
  -H "Content-Type: application/json" \
  -H "x-api-key: <YOUR_KEY>" \
  -d '{"title":"My Movie","year":2026,"rating":7.5,"genres":["Drama"],"plot":"Short plot..."}'

Endpoints yang ada

  • GET /v1/health (public)
  • GET /v1/movies (public)
  • GET /v1/movies/:id (public)
  • POST /v1/movies (protected)
  • PUT /v1/movies/:id (protected)
  • DELETE /v1/movies/:id (protected)
